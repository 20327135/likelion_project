# 🦁 likelion_project

멋쟁이사자처럼 프론트엔드 미션으로 진행한  
**아기사자 명단 인터랙티브 대시보드 프로젝트**입니다.

HTML, CSS, JavaScript를 활용하여  
자기소개 카드 대시보드를 제작하고,  
사용자가 직접 아기사자 정보를 추가/삭제할 수 있도록 구현했습니다.

## 👤 담당 역할

저는 프론트엔드 파트로 참여하여  
HTML, CSS, JavaScript를 활용한 자기소개 대시보드 웹페이지를 제작했습니다.

사용자가 한 페이지 안에서 아기사자들의 정보를 쉽게 확인하고,  
직접 추가 및 삭제까지 할 수 있는 인터랙티브한 구조를 구현하는 역할을 맡았습니다.

### 주요 담당 업무

- 전체 페이지 레이아웃 구성
- 자기소개 요약 카드 UI 제작
- 상세 자기소개 카드 영역 구성
- 반응형 CSS 레이아웃 구현
- JavaScript를 활용한 카드 추가 / 삭제 기능 구현
- 파트별 필터 기능 구현
- 빈 상태 UI 처리 및 동적 화면 전환 구현
- 요약 카드 클릭 시 상세 카드 자동 스크롤 구현

---

## 📅 1주차

## HTML / CSS를 활용한 자기소개 카드 제작

1주차에는 기본적인 자기소개 카드의 구조를 만들었습니다.

### 구현 내용

- 프로필 이미지
- 이름
- 소속 파트
- 한 줄 소개
- 관심 기술 배지

를 포함한 요약 카드를 HTML과 CSS로 구현했습니다.

카드가 단순한 정보 나열이 아니라  
시각적으로 보기 좋고 정돈된 형태가 되도록  
카드형 UI 디자인에 집중했습니다.

또한 내가 작성한 자기소개 카드는  
다른 카드들과 구분될 수 있도록  
테두리와 그림자를 활용해 강조했습니다.

---

## 📅 2주차

## CSS Grid와 반응형 레이아웃 구현

2주차에는 여러 장의 자기소개 카드를  
대시보드 형태로 배치하는 작업을 진행했습니다.

### 구현 내용

- 모바일 : 1열
- 태블릿 : 2열
- 데스크톱 : 3열

로 자동 변경되는 반응형 레이아웃을 구현했습니다.

또한 화면 하단에는  
각 아기사자의 상세 자기소개 카드가  
1열 리스트 형태로 자연스럽게 배치되도록 구성했습니다.

이 과정에서

- Grid Layout
- Flexbox
- Media Query

를 활용하여 화면 크기에 따라  
레이아웃이 무너지지 않도록 설계했습니다.

---

## 📅 3주차

## JavaScript를 활용한 인터랙티브 대시보드 구현

3주차에는 JavaScript를 활용하여  
정적인 화면을 동적으로 변경할 수 있도록 구현했습니다.

### 주요 기능

### 1. 아기 사자 추가 기능

폼(Form)을 통해

- 이름
- 파트
- 관심 기술
- 자기소개
- 연락처

등의 정보를 입력하면  
새로운 카드가 자동으로 생성되도록 구현했습니다.

---

### 2. 마지막 아기 사자 삭제 기능

삭제 버튼을 누르면  
가장 마지막에 추가된 카드가 제거되고  
총 인원 수도 함께 갱신되도록 만들었습니다.

---

### 3. 총 인원 표시 기능

현재 등록된 아기사자의 수를  
"총 n명" 형태로 실시간 표시하도록 구현했습니다.

---

### 4. 파트별 필터 기능

Frontend / Backend / Design

중 원하는 파트를 선택하면  
해당 파트의 카드만 화면에 보이도록 구현했습니다.

---

### 5. 빈 상태 UI 처리

등록된 아기사자가 0명이 되었을 때

"아직 등록된 아기사자가 없습니다."

라는 안내 문구가 자연스럽게 나타나도록 구현했습니다.

---

### 6. 요약 카드 클릭 시 상세 카드로 스크롤

상단 요약 카드를 클릭하면  
해당 인물의 상세 카드 영역으로  
자동 스크롤되도록 구현했습니다.

페이지 이동이 아닌  
같은 페이지 내 부드러운 이동 방식으로 처리했습니다.
